HTML: Structure of a Web Page

HTML (HyperText Markup Language) is the foundation of all websites. It defines content structure such as headings, paragraphs, links, images, forms, and section organization.

Basic Web Page Structure

A normal HTML page includes a <head> section for metadata and a <body> section for visible content. Semantic tags like <header>, <main>, and <footer> improve readability and accessibility. 

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My HTML Example</title>
  </head>
  <body>
    <h1>Welcome to My Page</h1>
    <p>This paragraph explains the main topic.</p>
    <a href="https://example.com">Visit Example</a>
    <img src="photo.jpg" alt="Descriptive text" />
  </body>
</html>

Common HTML Tags

  • <h1>...<h6> for headings and topic hierarchy
  • <p> for paragraphs and readable text blocks
  • <a> for hyperlinks between pages or websites
  • <img> for images with alternative text
  • <ul> <li> for bullet lists

Semantic HTML and Accessibility

Semantic tags help browsers, search engines, and assistive technologies understand page meaning. Accessibility improves when labels, headings, and alternative texts are used correctly.

  • Use <header>, <main>, and <footer> for structure.
  • Use clear heading order from h1 to h3.
  • Add alt text for images.
  • Use meaningful link text instead of generic “click here”.

Extended Markup Example

<main>
  <article>
    <h2>Admission Support</h2>
    <p>This section explains student support services.</p>
    <ul>
      <li>Course counseling</li>
      <li>Document guidance</li>
      <li>Interview preparation</li>
    </ul>
  </article>
</main>

This example demonstrates semantic grouping and clear content hierarchy for professional pages.
